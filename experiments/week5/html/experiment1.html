Routing from Angular JS

Home | About | Contact


About the experiment

This experiment makes use of the routing module of Angular JS to inject the content in an Angular view based on the navigation links that a user clicks. To keep this experiment as most basic way of explaining routing in Angular JS, the default controller is used.

Code Snippet

Below is the function that is being used for routing: 

routingApp.config(['$routeProvider',
  function ($routeProvider) {
      $routeProvider.
        when('/home', {
            templateUrl: '../routes/Home.html',
        }).
        when('/about', {
            templateUrl: '../routes/About.html',
        }).
        when('/contact', {
            templateUrl: '../routes/Contact.html',
        }).
        otherwise({
            templateUrl: '../routes/Home.html'
        });
  }]);
