TV Shows Details with AJAX

About the Experiment

This experiment uses an AJAX call to TV Maze API and gets a JSONP response.

A user enters the title of the TV show he wants information about in the input box. It makes a call to the generated URL and the client recieves a JSONP response. The response is then rendered to display the respective details on the page.

Code Snippet

The code to make an AJAX call to the TV Maze API is specified below: 

$("#getTvShow").click(function () {
        var title = $("#title").val();
        $.ajax({
            url: "http://api.tvmaze.com/search/shows?q=" + title,
            datatype: "json",
            success: renderTVShows
        });
    });
