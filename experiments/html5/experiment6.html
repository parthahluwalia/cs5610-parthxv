Designing the Home Page - jQuery FlexSlider

Link to the Experiment Demo

Description

This experiment demonstrates how to design a responsive home page using CSS and add some effects using JavaScript. In the first part of the experiment I wrote about how I used CSS to style and design the page. In this part let's I'll give a brief note on JavaScript libraries that have been used to design the page.

The page has a "scroll down" icon, that when clicked, takes a user to the imporatnt links down below the page scrolling smoothly over the contents of the page. This has been possible using a robust library of JavaScript - jQuery.flexslider. This requires jQuery to be first installed in the page, for it to take its desired effect. We just have to specify the "smoothscroll" class for the respective link containing the anchor tag to the part of the page, and the flexslider does its part.

Code Snippet

<p class="scrolldown">
    <a class="smoothscroll" href="#portfolio"><i class="icon-down-circle">
</p>
