Generating Sentiment Score for Each Tweet

RHcloud Link to the Experiment

About the experiment

This experiment calculates a sentiment score for each of the tweet that has been fetched based on a user's input. The sentiment analysis is based on text analysis and computational linguistics to identify and extract subjective information in source materials. The respective score generated for each tweet is then displayed.

Code Snippet

var tweetSentiment = [];
    twitterClient.get('search/tweets', { q: searchText }, function (error, tweets, response) {
        var status = tweets.statuses;
        for (var i in status) {
            tweetSentiment.push({
                tweet: status[i].text,
                sentiment: sentiment(status[i].text)
            });
            
        }
        res.json(tweetSentiment);
