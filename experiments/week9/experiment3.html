About the experiment

The app supports signing in through a user's Twitter Account using the PassportJS' twitter strategy. The module provided for this is passport-twitter, which ensures authentication to a user's twitter account while logging in. The twitter strategy uses the twitter app's consumer key, consumer secret and a callback URL to ensure authentication of a user.
