implementing users/search Twitter API

RHcloud Link to the Experiment

About the experiment

In order to search people in the network, a person could go to the "People" tab in the app and search for a user using their twitter username, a user's name, etc. and the results will be shown in the same view.

Code Snippet

app.get('/user-search/:user', function (req, res) {
    var user = req.params.user;
    twitterClient.get('users/search', { q: user }, function (error, users, response) {
        res.json(users);
    });
});
