RHcloud Link to the Experiment

About the experiment

This experiment demonstrates how to post a tweet from an app extending the study done in last week's last experiment. The great twitter API has made this provision avaliable to Twitter developers by providing "statuses/update" as a POST method. Hence this updates the authenticating user’s current status, also known as tweeting.

Code Snippet

 $scope.postTweet = function (tweetText) {
        $http.post('/wk7-exp5/' + tweetText)
            .success(function (response) {
                console.log(response);
            });
    };
