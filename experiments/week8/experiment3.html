About the experiment

This experiment demonstrates the sentiment of a subject i.e. how a subject of interest is feeling at the moment. The sentiment has been evaluated on the tweets fetched about the subject of concern. A comparative score of at most the last 15 tweets posted about the subject is calculated and a sentiment is then calculated based on the current score. The sentiment can change based on how frequently the tweets are posted about that subject.

if (0 < comparative && comparative < 0.15) {

                    $scope.phraseSentiment = "fantabulous"

                    $scope.phraseSentiment = "spectacularly out of the world"
                } else if (comparative === 0) {
                    $scope.phraseSentiment = "okay"
                } else if (-0.1 < comparative && comparative < 0) {
                    $scope.phraseSentiment = "not that good"
                } else if (-0.25 < comparative && comparative <= -0.1) {
                    $scope.phraseSentiment = "lousy"
                } else if (-0.55 < comparative && comparative <= 0.25) {
                    $scope.phraseSentiment = "awful"
                } else if (comparative <= -0.55) {
                    $scope.phraseSentiment = "awfully bad"
