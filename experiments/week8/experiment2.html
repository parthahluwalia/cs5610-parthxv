About the experiment

This experiment lays out the design of the home page of my project - A Sentiment Analysis based Social Network. This is intended to be a single page application so that a large number of features of the app can be accessed on the same page. It uses Bootstrap based styling in combination with many jQuery effects like easing, isotope, localscroll and nicescroll.

The layout of the app could be viewed and accessed at the link given above. Various other functionalities have been implemented as a part of future experiments.
