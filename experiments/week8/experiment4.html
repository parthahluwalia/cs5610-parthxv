Adding Contact Us and Maps in the App

RHcloud Link to the Experiment

About the experiment

This experiment adds the "Contact Us" and "Google Maps" in the app, in order for the user to submit a query to contact us and view us on Maps. This is minimalistic at this point and has no back-end script associated with it as of now.

Code Snippet

var mapOptions = {
        // How zoomed in you want the map to start at (always required)
        zoom: 11,
        // The latitude and longitude to center the map (always required)
        center: new google.maps.LatLng(40.6700, -73.9400), // New York
        // How you would like to style the map.
        // This is where you would paste any style found on Snazzy Maps.
        styles: [{ featureType: "all", elementType: "all", stylers: [{ invert_lightness: true }, { saturation: 10 }, { lightness: 30 }, { gamma: 0.5 }, { hue: "#1C705B" }] }]
    };
