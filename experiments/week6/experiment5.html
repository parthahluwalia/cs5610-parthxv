About the experiment

This experiment also uses modules browserify and sentiment to enhance the basic sentiment analysis application in a way that a user may persist the analyzed phrase in his app, and can also remove the analyzed phrase from his app by clicking on the remove button.

The controller sends the respective http get, post or delete request, and the modification is actually done at the server side.

Code Snippet

app.get('/experiment5/api', function (req, res) {
    res.json(analyzedSentiments);
});
app.post('/experiment5', function (req, res) {
    analyzedSentiments.push(req.body);
    res.json(analyzedSentiments);
});
app.delete('/experiment5/:index', function (req, res) {
    var index = req.params.index;
    analyzedSentiments.splice(index, 1);
    res.json(analyzedSentiments);
});
