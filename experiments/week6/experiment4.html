About the experiment

This experiment uses various modules available for node.js like browserify and sentiment and uses them to create a very basic sentiment analysis application that could analyze the sentiment of the phrase entered by a person. The sentiment score is calculated based on the polarity assigned to a word by the AFINN based word dictionary.

A user may enter any phrase and the app will display the score calculated for the phrase along with the words matched in the AFINN dictionary, upon clicking the "Analyze" button. Since browsers don't have the require method defined, Browserify lets us write code in the same way that we would it in Node, helping us in clear segregation of code.

Code Snippet

var sentiment = require('sentiment');
app.controller("SentimentController", function ($scope, $http) {
    $scope.evaluateSentiment = function () {
        var phrase = $scope.phrase;
        console.log("Analyzing phrase: " + phrase);
        var analysis = sentiment(phrase);
        $scope.analysis = analysis;
    }
});
