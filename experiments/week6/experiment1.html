About the experiment

This experiment is the most basic experiment in node.js that uses the index.html file in the public folder by default to load when it sees the RHcloud home path. The express app will use the statement - app.use(express.static(path.join(__dirname, '/public'))) - in order to achieve this functionality.

Code Snippet

var express = require('express');
var path = require('path');
var app = express();
var bodyParser = require('body-parser');
var multer = require('multer');
app.use(express.static(path.join(__dirname, '/public')));
