About the experiment

This experiment combines node.js and AngularJS in order for them to interact with each other. It uses the controller to make an http get request to the simple sentiment API created in the last experiment and renders the JSON response and show it as a table.

Code Snippet

app.controller("SentimentController", function ($scope, $http) {
    $http.get('/experiment2/api/sentiment')
        .success(function (response) {
            $scope.sentiments = response;
        });
});
