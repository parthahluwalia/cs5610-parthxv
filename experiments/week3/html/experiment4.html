Grid Animation with jQuery - Fading out the Images

About the experiment

This is a 3 experiment series walkthrough (combined in Experiments 3, 4 and 5) showcasing the usage of $.delay() function,jQuery easing plugin and CSS transforms to animate some of the thumbnails in the grid.

The final outcome of the experiment is intended to be this:
Upon clicking the Animate button, the thumbnails disappear randomly with a ripple effect. After all the thumbnails have faden away, they will quickly storm back into their original places in a cool 3D way.

Now, in the second experiment of this 3-experiment walkthrough, the ripple effect is created to display the fading out of thumbnails with style using Math.random() function followed by application of the delay() function. While the thumbnails are fading out a step function is used to apply some transfroms.

Code Snippet

The following code is being used to generate the ripple effect: 

        $(".animate").on("click", function(){
        //fading out the thumbnails with style
            $("img").each(function(){
                d = Math.random()*1000; //1ms to 1000ms delay
                $(this).delay(d).animate({opacity: 0}, {
                    //while the thumbnails are fading out, we will use the step function to 
                    //apply some transforms. variable n will give the current opacity in the animation.
                    step: function(n){
                        s = 1-n; //scale - will animate from 0 to 1
                        $(this).css("transform", "scale("+s+")");
                    }, 
                    duration: 1000, 
                })
            })
        })
