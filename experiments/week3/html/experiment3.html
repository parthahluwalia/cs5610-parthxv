Grid Animation with jQuery - Build Image Grid

About the experiment

This is a 3 experiment series walkthrough (combined in Experiments 3, 4 and 5) showcasing the usage of $.delay() function,jQuery easing plugin and CSS transforms to animate some of the thumbnails in the grid.

The final outcome of the experiment is intended to be this:
Upon clicking the Animate button, the thumbnails disappear randomly with a ripple effect. After all the thumbnails have faden away, theyback into their original places in a cool 3D way.

So, in this first experiment of the 3-experiment walkthrough, 50 random pictures will be fetched and displayed in a grid of 600x300px dimension.

Code Snippet

The following code is being used to generate the grid of 50 images: 

for (var i = 1; i <= count; i++) {
        images += '';
    }
    //Append images to the grid
    $(".grid").append(images);
