Grid Animation with jQuery - Storming back the Images

About the experiment

This is a 3 experiment series walkthrough (combined in Experiments 3, 4 and 5) showcasing the usage of $.delay() function,jQuery easing plugin and CSS transforms to animate some of the thumbnails in the grid.

The final outcome of the experiment is intended to be this:
Upon clicking the Animate button, the thumbnails disappear randomly with a ripple effect. After all the thumbnails have faden away, they will quickly storm back into their original places in a cool 3D way.

After the thumbnails have all faden away, a cool 3D animation style will be applied to storm them back into their original place. This will use the delay() and step() functions to translate and rotate the images along the axis.

Code Snippet

The following code is the storm() function that is being used to bring back the images in 3D style: 

    function storm() {
        $("img").each(function () {
            d = Math.random() * 1000;
            $(this).delay(d).animate({ opacity: 1 }, {
                step: function (n) {
                    //rotating the images on the Y axis from 360deg to 0deg
                    ry = (1 - n) * 360;
                    //translating the images from 1000px to 0px
                    tz = (1 - n) * 1000;
                    //applying the transformation
                    $(this).css("transform", "rotateY(" + ry + "deg) translateZ(" + tz + "px)");
                },
                duration: 3000,
                //Some easing fun. Comes from jquery easing plugin
                easing: 'easeOutQuint',
            })
        })
    }
