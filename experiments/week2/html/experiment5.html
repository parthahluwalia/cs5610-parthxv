About the Experiment

This experiment demonstrates how one can beutify their pages by using Bootswatch. The page is designed in order to view various details about the upcoming NFL Super Bowl. The NavBar here is customized in a way to provide convinent links to NFL like their Game Center, Predictions, or by clicking the drop down to navigate to the champions' pages like their respective profile, stats and schedule. A search bar is also encapsulated in the NavBar that can be customized to provide search functionality to a user.

A visitor to this web page can also post a comment about his favorite team amongst the two. In order to store and retrieve the comments, the concept of local storage provided by HTML5 has been used. A user may wish to provide his name while posting a comment and select his favourite team. As and when a new comment is posted, that comment is retrieved from the local storage and displayed along with the previously posted comments.

                function display() {
                    var commentBox = document.getElementById("allComments");
                    var comments = "";
                    for (var i = 0; i < sessionStorage.length; i++) {
                        var propertyName = sessionStorage.key(i);
                        var fetchedComment = sessionStorage.getItem(propertyName);
                        comments = comments + fetchedComment + "<br /> <br />";
                    }
                    commentBox.innerHTML = comments;
                }
