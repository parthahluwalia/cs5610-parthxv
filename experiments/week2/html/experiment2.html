Responsive Images


Change your browser width to see how the image adjusts itself to different browser window sizes.

About the experiment

This experiment the awesome combination of srcset and sizes attributes of img element to enable to enable an image to render itself according to the different browser sizes, targetting the screens of all the devices at different widths.

The srcset specifies various breakpoints (which are actually different browser widths) along with the source image, and the browser automatically picks up that image when its width is less than the specified breakpoint. The sizes attribute specifies the default length/size of the image that will be picked up when none of the specified breakpoints match.

Code Snippet

The srcset attribute is used as: 

            <img src="http://respimg.cdnconnect.com/on-a-grid/yacht_race@fallback.jpg"
                     srcset="http://respimg.cdnconnect.com/on-a-grid/yacht_race@mobile.jpg 320w,
                             http://respimg.cdnconnect.com/on-a-grid/yacht_race@wide-mobile.jpg 480w,
                             http://respimg.cdnconnect.com/on-a-grid/yacht_race@tablet.jpg 768w,
                             http://respimg.cdnconnect.com/on-a-grid/yacht_race@desktop.jpg 1024w,
                             http://respimg.cdnconnect.com/on-a-grid/yacht_race@hires.jpg 1280w"
                     sizes="65vw"
                     alt="A yacht race" />
