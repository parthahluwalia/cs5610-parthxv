Navigating to a User's Profile

RHcloud Link to the Experiment

About the experiment

When a user runs a search for people or views his twitter followers/friends, he might want to navigate to a user's profile. They can do so by clicking on a user's name in the respective results. As a user clicks on another user's profile, the respective user's data will be fetched from Twitter and rendered according to the app to be displayed to the user.

The data that will be rendered includes - Tweets, Twitter Friends and Twitter Followers.
