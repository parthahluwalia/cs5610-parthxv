About the experiment

When a user has logged in to his profile, one of the tabs that will be displayed to them is their twitter followers i.e. people who are following them on Twitter. In order to implement this another REST API of Twitter - followers/list - has been used, so that as the logged in user clicks on "Twitter Followers" tab, they get their the people they are following on Twitter.

Code Snippet

twitterClient.get('followers/list', { user_id: twitterId }, function (error, followers, response) {
        //console.log(followers);
        res.json(followers);
    });
