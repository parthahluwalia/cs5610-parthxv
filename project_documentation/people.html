People in the Network

You can search users in the network and watch their recent sentiments by navigating to this view of the app.

As the network is built on Twitter, it uses the users/search Twitter API making GET requests signed with OAuth.

GET users/search

Provides a simple, relevance-based search interface to public user accounts on Twitter. Try querying by topical interest, full name, company name, location, or other criteria. Exact match searches are not supported.
