User Profile

A user's profile (not of the current logged in user) shows the recent timeline of a user. At the top of their page a section is dedicated to show thier description and links / buttons to the useful information about them.

This section includes user's name, twitter screen name and description. This also contains links to show their tweets, Twitter friends and Twitter followers in the view below, and a button (if a user is logged into his account) to Monitor / Unmonitor a user.

The section below, the timeline of the user is set to show the most recent Analysis of how the person is feeling at the moment (which is can be viewd in detail by clicking the "Watch Analysis" button) and the recent tweets posted by the person, by default.

The data has been gathered from a bunch of methods in REST API of Twitter.

GET users/show

Returns a variety of information about the user specified by the required user_id or screen_name parameter. The author’s most recent Tweet will be returned inline when possible.

Click here for details of this method.

GET statuses/user_timeline

Returns a collection of the most recent Tweets posted by the user indicated by the screen_name or user_id parameters.

Click here for details of this method.

GET friends/list

Returns a cursored collection of user objects for every user the specified user is following (otherwise known as their “friends”).

Click here for details of this method.

GET followers/list

Returns a cursored collection of user objects for users following the specified user.

Click here for details of this method.
