AJAX

This week I performed AJAX and Anguar JS experiments using Web APIs. I have used TV MAZE API in order interact and get data from the Web.

The first experiment I gather details of a TV Show based on the name/title that user enters. Upon gathering the data, I tweak in the HTML tags to render that data in the HTML page. This experiment helped me understand how to handle AJAX calls to a Web API through jQuery and then render the results by creating HTML elements from jQuery.

In the second experiment, I enhanced the previous experiment to incorporate the local storage concept in HTML5. The last search results would be stored in the local storage which could be retreived upon clicking the respective button.
