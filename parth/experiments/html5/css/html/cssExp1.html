About the experiment and Code Snippets

This experiment shows how one can implement an inline style or an internal style sheet in CSS.

The inline style CSS can be implemented by using style attribute corresponding to a tag where the style has to be applied. This is shown in the snippet below: 

                    <div style="background-color: #00ffff; width:100%; height: 100px; text-align:center; line-height:90px">
                        <header>
                            <h2 style="font-style: oblique">Header Content</h2>
                        </header>
                    </div>

The internal style sheet, on the other hand can be implented by using <style>tag. The id and class attributes can then be accessed using their corressponding CSS notations. The snippet can be seen below: 

                    <style>
                        #westDiv {
                            float: left;
                            background-color: #2bac75;
                            width: 15%;
                            height: 400px;
                            text-align: center;
                        }

                        #west {
                            line-height: 15em;
                            transform: rotate(90deg);
                            transform-origin: left top 30;
                        }
                    </style>
