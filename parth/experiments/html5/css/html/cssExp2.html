Code Snippet

The external style sheet can be linked to an HTML document by using link tag as: 

        <link rel="stylesheet" type="text/css" href="../css/experiment2.css" />

The styling can then be applied just as in an internal style sheet using various norms of CSS. 

         #button-box { 
            height: 100px; 
            width: 100%; 
            position: relative; 
            float: left; 
            background: #353131;
        }
        .button {
            position: relative;
            top: 36%;
            left: 40%;
        }
