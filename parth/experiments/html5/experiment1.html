1. <aside> tag

The HTML <aside> element represents a section of the page with content connected tangentially to the rest, which could be considered separate from that content.

3. <dialog> tag

Represents a dialog box or an interactive window. The open attribute indicates a dialog is active and available for interaction. When the open attribute is not set, it shouldn't be shown to the user.
