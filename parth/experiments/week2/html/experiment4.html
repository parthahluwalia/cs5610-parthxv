About the Experiment

This experiment demonstrates how to use Bootstrap to build web pages. I have utilized the key peices of Bootstrap infrastructure to style the page. These peices include styling using key classes like jumbotron, table, table-responsive, navbar, success, warning, btn, etc.

You can enter the NFL Game Center to watch the matchup statastics for the teams playing in Super Bowl. You can also click on a corresponding team button (styled using btn, btn-primary, btn-block) that would land you to their nfl page. Similarly in order to view the respective passing leader's profile, one can click on his name link (styled using btn, btn-link). The table is made responsive using the table-responsive class in Bootstrap. This is done by wrapping .table class in .table-responsive class to make them scroll horizontally on small devices (under 768px).
