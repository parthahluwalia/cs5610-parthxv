About the experiment

This experiment demonstrates how to build a basic responsive web page using media queries and and providing the feature to switch navigation for the devices with less screen width.

Media queries help us combine a series of grabs, thus enabling the impact of transitions between different screen sizes to be appreciated fully. And since scaling down to a small screen could become unreadable and hard to click, we may want to also switch the navigation to enable a user to access it more easily.

Code Snippet

The following is the media query I wrote for this page: 

        @media screen and (max-width: 566px) {
            #skipTo {
            display: block;
            }
            header nav, #main, aside {
            float: left;
            clear: left;
            margin: 0 0 10px;
            width: 100%;
            }
            header nav li {
            margin: 0;
            background: #efefef;
            display: block;
            margin-bottom: 3px;
            }
            header nav a {
            display: block;
            padding: 10px;
            text-align: center;
            }
        }
