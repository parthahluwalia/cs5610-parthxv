About the experiment

This experiment demonstrates the simplest way to create a responsive design, yourself, with a simple CSS.

It uses simple div blocks tied to each other using the float attribute of CSS.

Code Snippet

The float attribute can be used as: 

            .lipsum {
                float: left;
                margin: 5px;
                padding: 5px;
                width: 300px;
                height: 300px;
                border: 1px solid black;
            }
