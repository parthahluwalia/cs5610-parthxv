The What Is Next Utility

The What Is Next utility is a convenient helper utility for the Story Utility.

The What Is Next utility is intended to be used as the first slide in a Story Utility slide show.

The What Is Next utility will display the list of titles present in the story file for the slide show. Furthermore each title will actually be a link to its url and the url will be opened in a new tab. This is convenient if you wish to show one particular slide directly.

To avoid redundancy, if the first slide in the slide show is a slide that references whatisnext.htm then this slide will be omitted from automatically generated list of titles.

Call the What Is Next utility via: 

.../story/whatisnext.htm?url=...

The url= query parameter gives the relative url to the story text file to be processed.

Actually, url= is optional since the utility will assume that an unnamed query parameter is the url.

If you wish to provide comments and more information on a What Is Next style page then use the more powerful Links utility instead.
