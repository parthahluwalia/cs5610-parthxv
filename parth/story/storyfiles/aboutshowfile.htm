The Show File Utility

The Show File utility is an important helper utility for the Story Utility.

The Show File utility will display all or part of a text file on a web page. The utility will HTML-encode all characters that might interfere with proper display and will automatically enclose the resulting text in a <pre> tag pair.

This utility is especially helpful for showing data files and programming source code. The fact that one may show selected lines in a file makes it possible to document and discuss portions of a large source code file.

Call the Show File utility via: 

.../story/showfile.htm?url=...&start=...&stop=...

The url= query parameter gives the relative url to the text file to be shown.

Actually, url= is optional since the utility will assume that an unnamed query parameter is the url.

The optional parameters start= and stop= give the lines in the file to start with and stop at. We follow the conventions:

  • The first line in the file has index 1.
  • The parameter start is inclusive.
  • The parameter stop is exclusive.

If one or both of these two parameters are omitted, the utility will provide the lines in the file that make sense.
