The Links Utility

The Links utility is a convenient helper utility for the Story Utility.

The Links utility is intended to be used either as the first slide in a Story Utility slide show or as an independent page that provides access to all links in a story file and can display additional titles, headings, and comments.

In short, the Links utility is fancy version of the What is Next utility.

The Links utility will display the titles present in a story file and each title will be a link to its url. The url will open in a new tab. In addition, the Links utility can display additional titles, headings, and comments by placing the information needed in lines of the story file that begin with blank or tab. These lines are ignored by the Story Utility and therefore can be used by the Links utility to provide additional content.

The additional content used by the Links utility is placed on lines that begin with at least one blank or tab. Then the content is introduced by one of the following tags:

  • !title=...
  • !pagetitle=...
  • !comment=...
  • !heading=...
  • !br
  • !hr

Let us explain how to use these tags.

The tag !title=... is used to set the title that is shown in the browser tab for the page. If this tag is omitted, then the browser tab will show the word “Links”.

The tag !title=... may be used only once and may not include any HTML markup. The other tags may be used multiple times and may contain valid snippets of HTML.

The tag !pagetitle=... is used to add one or more titles to the top of the web page. These page titles will be bold, centered, and in 140% font size.

The auto-generated titles/links associated with the url’s in the story file are displayed in bold with 125% font size.

Any additional comments and headings will be displayed in italic with 125% font size. Comments and headings must be entered on a single line in the story file but may render using multiple lines on the page. In that case, the rendered lines will be justified.

The tag !comment=... is used to enter a comment. A comment has no indentation, that is, it is flush left.

The tag !heading=... is used to enter a heading. A heading has left indentation of 3%.

In other words, a comment and a heading are the same except for the issue of indentation.

The tag !br tag is used to enter a vertical line break via the HTML code <br />.

The tag !hr tag is used to enter a horizontal rule via the HTML code <hr />.

The main content of a story file consists of url’s together with titles for these url’s. By interweaving the comment, heading, br, and hr tags with this data, one can provide significantly more information and formatting than by supplying simply a list of titles/links.

See the following Mini Demo for an example of an introductory page created with the Links utility.

Call the Links utility via: 

.../story/links.htm?url=...

The url= query parameter gives the relative url to the story text file to be processed.

Actually, url= is optional since the utility will assume that an unnamed query parameter is the url.
